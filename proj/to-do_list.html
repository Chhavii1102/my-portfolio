To-Do List

Description

This is a simple To-Do List application built using Python and Tkinter. It helps users add, mark, delete, and clear tasks easily. The application is designed to keep track of daily activities in a neat and organized way.

Features

  • Add new tasks to the list
  • Mark task as done
  • Deleted selected tasks
  • Clear all tasks with one click
  • Clean and user-friendly interface

Technology Used

  • Frontend : Tkinter(Python GUI)
  • Language : Python
