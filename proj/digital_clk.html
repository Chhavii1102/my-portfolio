Digital Clock

Description

This project is a Digital Clock made using Python and Tkinter .It display the current time in hours,minutes,and seconds,along with AM/PM format. It also shows the current date and day, updating automatically in real-time.

Features

  • Displays live time with hours,minutes,and seconds
  • Shows AM/PM format
  • Displays current date and day
  • Real-time automatic updates
  • Simple and easy-to-use interface

Technology Used

  • Python Tkinter
