Simple Notepad

Description

This is a Simple Notepad Application built using Python and Tkinter. It provides a lightweight and user-friendly interface for creating,editing and saving text files.
The application is designed to mimic basic features of a standard notepad while being simple and easy to use.

Features

  • Open an existing text file
  • Save file with updated content (Ctrl+S shortcut supported)
  • Create new text content and edit it freely
  • Exit the application with confirmation(Ctrl+Q)
  • Simple and clean user interface
  • Keyboard shortcuts for faster use:
    • Open -> Ctrl+O
    • Save -> Ctrl+S
    • Exit -> Ctrl+Q

Technology Used

  • Python - Programming Language
  • Tkinter - GUI Framework
