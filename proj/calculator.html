Simple Calculator

Description

This project is a basic calculator built using Python and Tkinter. It can perform different arithmetic operations and provides a clean interactive interface for users.
It is designed to make calculations simple,fast, and accurate.

Features

  • Perform basic operations:
    Addition(+),Subtraction(-),Multiplication(*),Division(/)
  • Percentage(%) calculation
  • Decimal number support(.) for precise calculations
  • Clear(C) button to reset the display
  • Equal(=) button to show final result
  • Error handling :
    • Display a clear error message like "Error" when dividing by zero(e.g.,10/0)
  • Attractive and easy-to-use GUI

Technology Used

  • Frontend & Logic : Python Tkinter
